Assignments


Assignment 1: Pong

In the first assignment you will learn how to make the game pong. This will make you learn the basic of programming. You will learn how to use variables, functions, classes and how monogame works. We expect you to have implemented these features for a passing grade.

  • Moving paddles and balls
  • Bounce interactions of the ball with the wall and the paddles
  • A scoring board and/or a live system
  • A start and end screen

You will get a higher grade for having clean code and/or for implementing new features in the game.

Assignment 2: Tetris

In the second assignment you will learn how to make the game Tetris. Here we will go a little more indepth into programming. You will learn how to use every kind of loop, how arrays and lists work and we will be building on the fundamentals of Pong. We expect you to have implemented these features for a passing grade.

  • A grid with falling blocks
  • The seven standard blocks of tetris
  • Being able to move the blocks and rotate them
  • Being able to destroy blocks when they form a horizantal full line
  • Seeing what the next block will be
  • A scoring system
  • The game must slowly becoming more difficult

You will get a higher grade for having clean code and/or for implementing new features in the game.

Assignment 3: TickTick

In the third assignment you will learn how to make the game TickTick. This is a platformer game just like Super Mario. we have already provided you a big part of the code and you will be adding more features to it. After this assignment you should be able too know your way around programming. You we use all the things you learned in the previous two assignment to compleet the assignment. We expect you to have implemented these features for a passing grade.

  • Make the player able to kill enemies by jumping on their heads
  • Implement a shooting feature
  • The sceen must move with the player when it moves

You will get a higher grade for having clean code and/or for implementing new features in the game.
